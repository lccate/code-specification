# 代码规范性
* 1 空格：在include后加空格  
```
#include <stdio.h>
```
前面推荐用四个空格代替Tab键（不同的公司要求不一样）  
* 2 大括号：有些时候if语句只有一句话，好多人就不写大括号，最好是只要有内容就写大括号，以后再添加语句的时候不容易发生错误  
* 3 符号位加减乘除等号两边加空格，增强代码可读性，看起来更工整  
```
a = b + c;
```
* 4 变量名尽量用英文单词或者英文单词的缩写，避免用a,b这种简单的名字  
常用的名字：  
sum 和  
product 积  
temporary\temp\tmp 临时的值  
flag 标识
